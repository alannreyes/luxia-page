Alert Acknowledgment System

Complete visibility into alert reception and response

ACK Type Description User Action Analytics Value
CONFIRMED User acknowledged the alert Pressed "Understood" button Active engagement confirmed
TIMEOUT Alert auto-dismissed No interaction within timeout period Possible alert fatigue or away from device

Case Study: Antamina

Real requirements from Peru's largest copper-zinc mine

Current Situation

  • 3 Areas of Interest (AOIs), expanding to 6
  • Alert radii: 4/8/12 km and 8/16/32 km
  • ~2,500 workers need to receive alerts
  • Mix of employees and contractors
  • Requested SISMATE-like functionality

Their Ask

"Alerts should only reach devices that are physically inside the mine site, similar to how SISMATE works with cell broadcast."

EFC Alerta Solution

Phase 1: Android Deployment

  • Install EFC Alerta on worker devices
  • IT deploys via MDM (managed devices)
  • Workers download from Play Store (BYOD)

Phase 2: Expanded Coverage

  • iOS app for supervisors/management
  • Desktop alerts via PWA (control rooms)
  • Integration with fleet management

Phase 3: Analytics

  • ACK rate dashboards
  • Response time metrics
  • Coverage heat maps
